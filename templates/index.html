How do you open a new tab for files like *.cpp or *.h?

Have you ever tried to click on a *.txt file inside a browser? You'll have noticed that the browser will open the file inside a new tab (at least when you middle click on it). Well, have you tried the same action with *.cpp, *.h, or any other source files?

I'm not sure if you've ever tried this, but if that was the case, then for sure you'll have noticed that the browser this time does not conveniently open a new tab and show the file to you, but it offers the latter as a downloadable file. Depending on the situation this might be exactly what you want. But what if you'd still prefer a tab for such files? How would you handle that?

One possible solution is to convert these files to HTML and serve them; then the browser will open these for sure in their own tabs. And their is a very nice tool that can be handy for that task: pygments.org. If you've installed it you execute the following command:

pygmentize -f html -O full -o "cpp-source.html" "source.cpp"

It will take the source.cpp file and generate the cpp-source.html HTML file. Below this has been done for one *.cpp and one *.h file: Click on the links and the file will be opened in a new tab showing the corresponding source with even proper syntax highlighting.
